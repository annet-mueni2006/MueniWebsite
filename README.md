**Project Description**
My Journey is a personal portfolio website built to showcase my background, skills, and academic/technical projects as a Diploma in Information Technology student. The site presents a clean, easy-to-navigate overview of who I am, what I can do, and how to get in touch, and serves as a central place to direct potential employers, collaborators, and academic reviewers to my work.
Technologies Used
•	HTML5 — page structure and content
•	CSS3 — styling, layout, and responsive design

**Features Implemented**
•	Home / Index page introducing the site and site owner
•	About section describing background and interests
•	Skills section highlighting technical competencies
•	Projects section showcasing completed work
•	Resume section with academic and professional details
•	Contact section for reaching out
•	Responsive, consistent styling across all pages

**Installation Instructions**
1. Clone the repository:
git clone https://github.com/annet-mueni2006/my-journey.git
2. Navigate into the project folder:
cd my-journey
3. Open the index.html file in your preferred web browser to view the site (double-click the file, or right-click and choose "Open with" your browser).
